Upload datasets, invite model vendors, and compare results. Your data never leaves.

## How It Works

1. **Deploy** a secure workspace on your machine with a single command
2. **Define** a use case — select datasets, set metrics, invite contributors
3. **Evaluate** — vendors submit models that run inside your infrastructure
4. **Compare** results across models with built-in reporting

## Who Is This For

**Data owners** — set up use cases, control access, evaluate vendor models on real production data before making buy-or-build decisions.

**Data scientists** — connect to use cases, train and fine-tune models on client infrastructure, submit results.

## Platform Components

- **Model Submission Interface** — vendors upload and manage model submissions
- **Secure Execution Layer** — deployed inside your infrastructure, runs all workloads locally
- **Evaluation & Reporting Engine** — tracks accuracy, latency, cost, and summarizes tradeoffs

All communication is TLS-encrypted. Only metadata and metrics leave your network. [Learn more about security](/environment-setup/configuration#security).
