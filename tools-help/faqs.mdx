# Frequently Asked Questions

Find answers to common questions about using Tracebloc.

## General Questions

### What is Tracebloc?
Tracebloc is a platform that enables secure and private AI model evaluation. It allows data owners to test vertical AI models from external vendors while maintaining data privacy.

### How does data privacy work?
- Data never leaves your infrastructure
- Model weights are encrypted
- No raw data sharing
- Secure communication channels
